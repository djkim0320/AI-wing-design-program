# 릴리즈 노트

## v1.0.1 (2026-02-08)

### 수정 사항

- 일부 환경에서 `WingDesignerAI.exe` 실행 시 발생하던 `Python.Runtime.Loader.Initialize` 오류를 우회하도록 실행 방식을 개선했습니다.
- 네이티브 데스크톱 창 초기화에 실패하면 앱이 기본 브라우저 모드로 자동 전환됩니다.
- 브라우저 모드 전환 시 종료 버튼이 있는 보조 창을 제공해 앱을 정상 종료할 수 있습니다.
## v1.0.0 (2026-02-08)

연구실 공유를 위한 첫 공개 릴리즈입니다.

### 포함 내용

- Windows 실행 패키지 (`WingDesignerAI.exe`)
- 앱 내 AI 제공자 설정 (Gemini/GPT/Claude/Grok)
- 3D 날개 뷰어 + STL/OBJ 내보내기
- 에어포일 및 공력 성능 분석 화면
- 히스토리 저장/불러오기/비교 기능
- 릴리즈 폴더 내부 사용자 설명서

### 패키지 경로

- `AI_Design_Program_Run`
